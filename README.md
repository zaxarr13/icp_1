# Система управления расписанием маршрутов (консольное приложение)

## Описание

Данное консольное приложение моделирует простую информационную систему управления расписанием маршрутов общественного транспорта.  
Программа работает с табличными данными на базе `pandas`, ведёт лог действий пользователя в текстовом файле и позволяет сохранять состояние «базы данных» в CSV.

Поддерживаются следующие виды транспорта:
- Поезд
- Автобус
- Электричка
- Трамвай

Пункты отправления/прибытия выбираются из фиксированного списка городов (Москва, Санкт-Петербург и др.).


## Основные возможности

- Генерация начального расписания маршрутов (случайные данные).
- Просмотр текущего расписания в виде таблицы.
- Добавление нового маршрута с проверкой корректности данных.
- Удаление маршрута по номеру.
- Поиск маршрутов:
  - по пункту отправления;
  - по пункту прибытия;
  - по типу транспорта.
- Покупка билетов:
  - уменьшение количества свободных мест;
  - поиск альтернативных маршрутов при отсутствии мест.
- Поиск маршрутов, находящихся в пути в заданный момент времени.
- Получение статистики:
  - среднее время в пути по видам транспорта;
  - средняя цена билета;
  - маршруты с заполненностью ниже заданного порога;
  - самый частый вид транспорта;
  - топ‑3 самых долгих маршрута.
- Сортировка маршрутов по длительности (возрастание/убывание).
- Фильтрация маршрутов по типу транспорта.
- Визуализация:
  - диаграмма средней продолжительности по видам транспорта (`diag1.png`);
  - диаграмма заполненности маршрутов (`diag2.png`);
  - график пунктов прибытия для выбранного города отправления (`gr1.png`).
- Сохранение текущего состояния данных в CSV-файл (`result.csv`).
- Логирование всех ключевых действий в файл `log.txt`.


## Используемые технологии

- **Язык программирования:** Python 3
- **Основные библиотеки:**
  - `pandas` — работа с табличными данными;
  - `numpy` — генерация случайных числовых данных;
  - `matplotlib` — построение графиков и диаграмм;
  - `datetime` — работа с датой/временем (для логов);
  - стандартные модули Python (`os`, `random` и др.).
